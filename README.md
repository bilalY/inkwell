# Inkwell

A Windows batch script that sets up folder structure templates for new Etsy digital product designs (planners, journals, printables).

## What It Does

Double-click `inkwell.bat`, answer a few prompts (product name, type), and get a ready-to-use project folder with consistent structure and template files.

## Usage

1. Download `inkwell.bat` to your working directory
2. Double-click it (or run from Command Prompt)
3. Follow the prompts

## Tech Stack

- Windows Batch (.bat)
- Zero dependencies
